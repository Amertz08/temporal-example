# Project Overview

This document outlines the project requirements and process.

## User Profiles

### Vehicle Owner
A vehicle owner is the state resident who wants to register their vehicle in order to get tags

### State Staffer
A state staffer is the state employee who will process vehicle registration applications and issue tags

## Process Steps
1. Vehicle Owner submits a vehicle registration application
2. State Staffer reviews the application
   1. Vehicle Owner will have to show proof of identity
   2. State Staffer will validate the vehicle registration application and proof of identity
3. Manufacturing request is sent to the license plate manufacturer
4. License plate is manufactured it will be sent to the Vehicle Owner's provided address
   1. Vehicle Owner will be notified of completion and estimated delivery date
5. Vehicle Owner will be notified of license plate delivery
